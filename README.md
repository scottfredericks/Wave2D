# Wave2D
A simple C++ simulation of the 2D Wave equation using SDL2

Dependencies:  
* SDL2:  
```sudo apt-get install libsdl2-dev```  
* make:  
```sudo apt-get install make```

To Use:  
  ```git clone https://github.com/scottfredericks/Wave2D```  
  ```cd Wave2D```  
  ```make```  
  ```./RunWave```
